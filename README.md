# Cross-Domain-Deception-Detection-via-Extended-DeCLaRatiVE-Stylometry
Cross-domain deception detection using 48 psychologically motivated  stylometric features across 6 theories (Distancing, Cognitive Load,  Reality Monitoring, Verifiability, IMT, IDT), 6 domains, and 4  classifiers (LR, LinearSVC, XGBoost, DistilBERT). BEP thesis — TU/e 2026.
